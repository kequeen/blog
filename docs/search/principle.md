## 主要流程
### 语义理解
如何理解用户的搜索 query，比如用户搜索，query = 北京天气怎么样,我们如何理解到用户的意图是要查询北京的天气
### 召回
确认哪些结果可以被召回，还是以query = 北京天气 举例，

### 排序
一般会分为粗排和精排


## 模块划分
其实主要也就分为三个模块：
### 信息采集
一般这一步就是我们所熟知的爬虫的工作过程，如何爬取互联网上所有的信息

### 离线建库
爬虫抓取到的信息，大多数是非结构化的，如果将这非结构化的数据转换为结构化的信息，供统一进行排序

### 在线检索
对用户搜索的 query 进行语义识别，改写、纠错、切词
