# 关于动态规划
动态规划相关的算法，其实是这些年面试最常见的高频题目了，其实也是相当考验个人能力  
有时候会很喜欢dp的思想，构建状态转移方程，就能够通过自顶向下地计算获取结果  
最核心的就是如何构造出动态规划方程  
其实最容易搞混的就是递归与动态规划的必然联系，应该是说动态规划其实是将大问题拆解为小问题，而需要优化的话，一般就是存储已经求解的子问题的答案，减少重复计算。一般其实采用自底向上计算
